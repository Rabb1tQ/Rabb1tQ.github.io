委托简单使用

定义委托：

 public delegate int CalculatorDelegate(int a, int b);

然后定义两个和委托类型和参数个数一样的方法：


static int Add(int a,int b)
           {
            return a + b;
           }

static int Sub(int a,int b)
           {
            return a - b;
           }

//创建委托对象，关联Add方法
            CalculatorDelegate objCal = new CalculatorDelegate(Add);
            //通过委托调用方法，而不是直接使用方法
            int result = objCal(10, 20);
            Console.WriteLine(result);

输出结果为：30

objCal -= Add;//断开当前委托所关联的方法Add

objCal += Sub;//重新关联一个方法Sub

result = objCal(30,10);

输出结果为：20

委托一般声明在类的外部，根据委托定义的函数原型编写需要的方法，委托在使用前必须实例化，实例化的时候需要指明默认的方法，委托调用方法，而不是直接调用方法，委托对象所关联的方法可以动态变化。
